# BAAMT – Behavioural Framework for Designing Advocacy Messaging

## Overview
BAAMT is a prototype decision-support tool designed to help advocates evaluate and improve advocacy messaging using behavioural science principles.

The framework translates behavioural insights into a structured assessment that helps identify strengths and weaknesses in advocacy communication strategies. By highlighting factors such as credibility, emotional resonance, clarity, and perceived efficacy, the tool helps advocates design messages that are more likely to resonate with audiences and motivate constructive action.

---

## How the Framework Works

The tool guides users through a structured evaluation process:

1. Users assess an advocacy message across a series of behavioural criteria.
2. Each criterion reflects an insight from behavioural science or communication research.
3. Responses generate a profile highlighting strengths and weaknesses in the message.
4. The tool provides insights into how the messaging strategy might be improved.

# BAAMT – Behavioural Framework for Designing Advocacy Messaging

## Overview
BAAMT is a prototype decision-support tool designed to help advocates evaluate and improve advocacy messaging using behavioural science principles.

## How It Works
Users answer a structured set of questions evaluating messaging across criteria such as:

- clarity
- credibility
- emotional resonance
- perceived efficacy
- social norms

The tool then generates an assessment highlighting strengths and potential weaknesses in the messaging strategy.
